Zilgraph

Zilgraph is an open source Zilswap Dashboard for visualizing ZilSwap activity on the Zilliqa blockchain.

It's providing XSGD and GZIL charts as well as other ZRC-2 tokens.

The Zilgraph front-end provides visualized data using elasticsearch, kibana and bokeh python library.
